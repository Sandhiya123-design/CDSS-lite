🏥 CDSS-Lite: Clinical Decision Support System

A simple web-based Clinical Decision Support System (CDSS) that helps assess patient risk using symptoms, vital signs, and comorbidities. It provides a rule-based risk evaluation and displays results in an interactive Streamlit dashboard.

✨ Features
Patient information input (age, weight, height, etc.)
Vital signs analysis (temperature, SpO₂, heart rate, blood pressure)
Symptom severity selection
Comorbidity detection
Rule-based risk scoring system
Risk classification (Low / Moderate / High)
Patient history storage (CSV)
Data visualization (graphs and trends)
🛠️ Technologies Used
Python
Streamlit
Pandas
NumPy
Matplotlib

🧠 How It Works
User enters patient data and symptoms
System calculates risk score using rule-based logic
Scores are mapped to risk levels
Output shows:
Risk Level
Recommended action
Visual graphs
Data is stored for history tracking
